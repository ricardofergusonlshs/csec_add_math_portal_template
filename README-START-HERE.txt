CSEC Add Math Portal template

This folder contains the setup files for your Vite + React + TypeScript project.

Files included:
- package.json
- vite.config.ts
- tsconfig.json
- tsconfig.app.json
- tsconfig.node.json
- index.html
- .env.example
- src/main.tsx
- src/index.css
- src/App.tsx

IMPORTANT:
1. Open src/App.tsx
2. Replace the placeholder content with your full app code from ChatGPT canvas
3. Rename .env.example to .env if needed
4. Fill in:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_PUBLISHABLE_KEY
   - VITE_TEACHER_PASSWORD

Then upload these files to GitHub or open locally in VS Code.
